SUMMARY (ENGLISH)
This project implements an automated data analytics pipeline for FIFA World Cup data analysis and projections for the 2026 tournament.

Key Components:
Data Storage: Amazon S3 with IAM-controlled access
ETL Processing: Manual ETL via Jupyter Notebooks and automated daily ETL using AWS Lambda
Visualization: Interactive Streamlit dashboard running on AWS EC2 instance

Deployment: GitHub repository with reproducible code

Technologies: Python, Pandas, Boto3, AWS S3, AWS Lambda, EventBridge, EC2, Streamlit, Plotly, GitHub
